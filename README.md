# Photolithography-CD-Optimization
Optimized photolithography process using JMP &amp; Six Sigma to eliminate 9.3% scrap. Resolved low Dose &amp; PEB_Temp issues by shifting parameters (Dose=35, PEB_Temp=130°C). Boosted process capability (Cpk) by +372% (0.44 to 2.075) and achieved 0.6608µm Critical Dimension (target 0.65).
